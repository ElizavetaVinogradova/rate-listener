1. Установите соединение с биржей криптовалют CoinBase через WebSocket (документация: https://docs.cloud.coinbase.com/exchange/docs/websocket-overview)
2. Подпишитесь на обновления котировок по трём инструментам: ETH-BTC, BTC-USD, BTC-EUR. Данные будут поступать через WebSocket по мере их появления на бирже.
3. Создайте таблицу "ticks" в MySQL со следующими полями: 
    timestamp (BIGINT) - Unixtime в миллисекундах, 
    symbol (VARCHAR) - название инструментов, 
    best_bid (DOUBLE) - цена предложения продажи, 
    best_ask (DOUBLE) - цена предложения покупки.
4. Установите соединение с MySQL и записывайте в таблицу "ticks" данные, полученные через WebSocket. 
Необходимо предусмотреть запись данных пакетами (батчами), а также повторные попытки (ретраи) в случае ошибки записи. 
Размер пакета, таймаут и максимальное количество попыток должны быть конфигурируемыми

5. Добавить Кафку, чтобы писать в нее тики, а потом уже читать из Кафки и сохранять в базу.
5.1 Добавить еще один main чтобы читатель и писатель Кафки были в разных бинарниках. Это важно для масштабируемости.
